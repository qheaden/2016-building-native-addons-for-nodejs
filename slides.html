Creating Native Modules for Node.js

What Are JavaScript Engines?

  • Provide an execution environment for JS code
  • Built in lower-level languages such as C/C++
  • Many variations are available today

Major JS Engines Today

  • SpiderMonkey - Mozilla Firefox
  • Chakra - Internet Explorer 9 through 11
  • ChakraCore - Edge
  • V8 - Google Chrome

These JS Engines Are Embeddable

  • Web Browsers (Of Course!)
  • Game Engines
  • Other Scriptable Tools
