Creating Native Modules for Node.js

What Are JavaScript Engines?

  • Provide an execution environment for JS code
  • Built in lower-level languages such as C/C++
  • Many variations are available today

Major JS Engines Today

  • SpiderMonkey - Mozilla Firefox
  • Chakra - Internet Explorer 9 through 11
  • ChakraCore - Edge
  • V8 - Google Chrome

These JS Engines Are Embeddable

  • Web Browsers (Of Course!)
  • Game Engines
  • Other Scriptable Tools

Node.js Provides A C++ API

Hello World Module C++ Code

                        
                                // hello.cpp

                                #include <node.h>

                                using v8::FunctionCallbackInfo;
                                using v8::Isolate;
                                using v8::Local;
                                using v8::Object;
                                using v8::String;
                                using v8::Value;

                                void GetHello(const FunctionCallbackInfo<Value>& args) {
                                    Isolate* isolate = args.GetIsolate();

                                    args.GetReturnValue().Set(String::NewFromUtf8(isolate, "Hello World!"));
                                }

                                void Initialize(Local<Object> exports) {
                                    NODE_SET_METHOD(exports, "getHello", GetHello);
                                }

                                NODE_MODULE(NODE_GYP_MODULE_NAME, Initialize)

Build Using Gyp

                        
                            {
                                "targets": [
                                    {
                                        "target_name": "hello",
                                        "sources": [ "hello.cpp" ]
                                    }
                                ]
                            }
                        
                    
                        
                            $ node-gyp init
                            $ node-gyp build

Import the Module

                        
                            // test.js

                            var helloModule = require('./build/Release/hello.node');
                            console.log(helloModule.getHello());
                        
                        
                            $ node ./test.js
                            $ Hello World!
